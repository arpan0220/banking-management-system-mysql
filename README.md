# Banking Management System using MySQL

## Overview
This project is a Banking Management System developed using MySQL.

It manages:
- Customers
- Accounts
- Transactions
- Loans
- Branches

## Features
- Deposit & Withdrawal Procedures
- Trigger for Negative Balance Prevention
- Transaction Tracking
- SQL Joins & Views

## Technologies Used
- MySQL
- SQL
- MySQL Workbench
